# QuickQR Generator 

A simple, lightweight, and responsive web application that generates QR codes from text or URLs instantly.

## Features
- Real-time Generation**: Create QR codes with one click.
- Responsive Design**: Works perfectly on desktop and mobile.
- Clean UI**: Minimalist and user-friendly interface.
- Built with**: Pure HTML, CSS, and JavaScript.

## How to Use
1. Clone the repository.
2. Open `index.html` in your browser.
3. Type any URL or text in the input field.
4. Click "Generate Code".

## Technologies Used
- HTML5
- CSS3 (Flexbox & Custom Styling)
- JavaScript (Vanilla JS)
- [QRCode.js Library](https://davidshimjs.github.io/qrcodejs/)
